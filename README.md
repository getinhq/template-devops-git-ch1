# Create and manage a Git repository

## Task
Initialize a Git repo, add a README, create a branch, commit changes, and push to GitHub.

## Starter (not finished)
This folder is a starting point only. You perform Git operations in **your fork** of this template.

## Your work
1. Use this repo (or your fork) locally: `git init` if needed, add files, commit.
2. Create a branch (e.g. `add-content`), make a change, commit.
3. Push to `origin` on GitHub.
4. Fill in `notes.txt` with the branch name and remote URL you used.

## Checklist
- [ ] At least two commits on a feature branch
- [ ] Remote `origin` points to your GitHub repo
- [ ] `notes.txt` documents what you did
